PARA LAS CITAS BIBLIOGRAFICAS

1. Cómo hacer citas bibliográficas Normas APA https://normas-apa.org/referencias/citar-pagina-web/

Ejemplo:
Carlos Sanchez (05 de febrero de 2020). ¿Cómo citar una Página Web?. Normas APA. Recuperado el 5 de noviembre de 2025 de https://normas-apa.org/referencias/citar-pagina-web/

2. Citas bibliográficas ejemplo de Iribel
- Las citas bibliográficas deben incluirse de la siguiente manera:

Apellido, Nombre (año). Titulo de la Página o del artículo (en negritas). Dirección electrónica (http://www.....) [Fecha de la consulta entre corchetes)]

Ejemplo:
CLEMPNER K., Julio y GUTIÉRREZ T., Agustín (2002). Administración y Ejecución de un Plan Estratégico de Tecnología de Información. Disponible en: http:... [Consultado: 20/03/2010]

- Si en la página web no consigue el autor o el año, entonces debe colocar para la cita el nombre completo de la organización que está publicando el texto que les interesa, generalmente el nombre de la página web.

Ejemplo:
Biblioteca de Empresas Públicas de Medellín (EPM) (2007). Las TIC. Disponible en: http://www... [Consultado: 02/11/2009]
